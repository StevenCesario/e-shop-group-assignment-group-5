# Verksamhetsbeskrivning
Vi skapar en e-shop där det säljs NES/SNES med tillbehör. Så som kontroller, sladdar och diverse spel. Vi riktar oss till alla åldrar, men mycket 80-talister och bakåt, de som hade en konsol själva samt växte upp med det.

## Typsnitt: SNES/NES Font / “pixligt” i Hero och eventuellt header och/eller footer men sen en “vanlig” läsbar trevlig font på resten av sidan och på produktsidorna.
## Tema: Konsoler/handkontroller NES/SNES - Retro
## Färgschema: Grå, Röd, Svart, Vit
## Känsla: Nostalgia, Flash, Joy, Excitement, Playful, Adventure
